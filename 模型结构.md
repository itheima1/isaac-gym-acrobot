# Acrobot 模型结构与空间定义

本文档详细记录了 Acrobot 任务的观测空间、动作空间定义，以及神经网络模型的输入输出结构。

## 1. 观测空间 (Observation Space)

模型需要感知的环境状态信息。

- **维度 (Shape)**: `(num_envs, 6)`
- **数据类型**: `float32`
- **详细构成**:
  | 索引 | 名称 | 物理含义 | 来源代码 |
  | :--- | :--- | :--- | :--- |
  | 0 | `sin_shoulder` | 第一关节角度的正弦值 | `torch.sin(shoulder_pos)` |
  | 1 | `cos_shoulder` | 第一关节角度的余弦值 | `torch.cos(shoulder_pos)` |
  | 2 | `sin_elbow` | 第二关节角度的正弦值 | `torch.sin(elbow_pos)` |
  | 3 | `cos_elbow` | 第二关节角度的余弦值 | `torch.cos(elbow_pos)` |
  | 4 | `shoulder_vel` | 第一关节（肩部）角速度 | `self.dof_vel[env_ids, 0]` |
  | 5 | `elbow_vel` | 第二关节（肘部）角速度 | `self.dof_vel[env_ids, 1]` |

- **代码位置**: 
  [acrobot.py](isaacgymenvs/tasks/acrobot.py) 中的 `compute_observations` 函数。
  ```python
  self.obs_buf[:, 0] = torch.sin(shoulder_pos)
  self.obs_buf[:, 1] = torch.cos(shoulder_pos)
  self.obs_buf[:, 2] = torch.sin(elbow_pos)
  self.obs_buf[:, 3] = torch.cos(elbow_pos)
  self.obs_buf[:, 4] = shoulder_vel
  self.obs_buf[:, 5] = elbow_vel
  ```

## 2. 动作空间 (Action Space)

模型输出的控制指令。

- **维度 (Shape)**: `(num_envs, 1)`
- **数据类型**: `float32`
- **控制方式**: `DOF_MODE_EFFORT` (力矩控制)
- **详细构成**:
  | 索引 | 名称 | 物理含义 | 控制范围 | 备注 |
  | :--- | :--- | :--- | :--- | :--- |
  | 0 | `shoulder_effort` | 施加在第一关节的力矩比例 | `[-1.0, 1.0]` | 最终力矩 = 输出值 * maxEffort (1.0) |

- **执行逻辑**:
  1. 模型输出一个 `[-1, 1]` 之间的浮点数（通常会经过 clip）。
  2. 程序将其乘以 `maxEffort` (1.0 N·m)。
  3. 结果作为力矩施加在 Joint 0 (Shoulder) 上。
  4. Joint 1 (Elbow) 是被动关节，不施加任何力矩。

- **代码位置**:
  [acrobot.py](isaacgymenvs/tasks/acrobot.py) 中的 `pre_physics_step` 函数。
  ```python
  # Only actuate joint 0
  actions_tensor[0::self.num_dof] = actions.to(self.device).squeeze() * self.max_push_effort
  ```

## 3. 神经网络模型结构

使用 PPO (Proximal Policy Optimization) 算法训练的 Actor-Critic 网络。

- **配置文件**: [AcrobotPPO.yaml](isaacgymenvs/cfg/train/AcrobotPPO.yaml)
- **网络类型**: MLP (多层感知机)
- **输入层**: 6 个神经元 (对应观测空间维度)
- **隐藏层**: 
  - 第一层: 256 个神经元, 激活函数 ELU
  - 第二层: 128 个神经元, 激活函数 ELU
- **输出层 (Actor)**: 1 个神经元 (对应动作空间维度，输出均值 mu)
- **输出层 (Critic)**: 1 个神经元 (输出价值 Value)

```yaml
  network:
    name: actor_critic
    separate: False
    mlp:
      units: [256, 128]
      activation: elu
```

## 4. 数据流总结

1. **Input (State)**: `[Sin1, Cos1, Sin2, Cos2, Vel1, Vel2]` (6维向量)
   ↓
2. **Neural Network**: `Input(6) -> Dense(256) -> Dense(128) -> Output(1)`
   ↓
3. **Output (Action)**: `Control Signal` (标量, 范围 -1.0 ~ 1.0)
   ↓
4. **Physics Engine**: `Torque = Action * 1.0 N·m` -> Apply to Joint 0
