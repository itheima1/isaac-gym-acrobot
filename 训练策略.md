# Acrobot 训练策略文档

本文档详细记录了 Acrobot (双连杆倒立摆) 任务的训练策略、奖励机制及关键参数配置。

## 1. 任务描述
目标是将一个双连杆系统（第一关节电机驱动，第二关节被动）从随机初始状态摆动并稳定在直立位置（两个关节角度均为 0）。

## 2. 关键参数配置

### 2.1 物理参数
- **连杆质量**: 0.070 kg (70g) / 每个连杆
  - *配置位置*: [acrobot.urdf](assets/urdf/acrobot.urdf)
- **电机力矩 (Max Effort)**: 5.0 N·m
  - *配置位置*: [Acrobot.yaml](isaacgymenvs/cfg/task/Acrobot.yaml)
  - *说明*: 之前尝试过 400.0 和 50.0 都过大导致飞车，2.0 略小，目前调整为 5.0 以提供足够的控制力。
- **摩擦力**: 阻尼 0.005, 干摩擦 0.01
  - *配置位置*: [acrobot.urdf](assets/urdf/acrobot.urdf)

### 2.2 时间参数
- **仿真步长 (dt)**: 0.0166s (60Hz)
- **子步数 (substeps)**: 2
  - *说明*: 每一次控制决策对应 2 次物理仿真步，即控制频率为 30Hz (0.0333s)。
- **最大回合长度 (Max Episode Length)**: 500 步
  - *对应时间*: 500 * 0.0333s ≈ 16.66 秒
  - *配置位置*: [acrobot.py](isaacgymenvs/tasks/acrobot.py) (`self.max_episode_length = 500`)

## 3. 奖励与惩罚机制 (Reward Function)

代码实现位于: [acrobot.py](isaacgymenvs/tasks/acrobot.py) 的 `compute_acrobot_reward` 函数。

### 3.1 奖励组成 (每一步)

总奖励 = `存活奖励` + `姿态奖励(Shoulder)` + `姿态奖励(Elbow)` + `速度惩罚`

1.  **存活奖励 (Survival Reward)**:
    - **值**: `+1.0`
    - **条件**: 只要当前步没有触发重置（没有倒下超过阈值），就获得此奖励。

2.  **姿态奖励 (Pose Reward)**:
    - **公式**: `exp(-2.0 * error^2)`
    - **说明**: 使用高斯核函数。当角度误差为 0 时，奖励最大为 `+1.0`；误差越大，奖励呈指数衰减。
    - **Shoulder**: `exp(-2.0 * shoulder_pos^2)` (最大 +1.0)
    - **Elbow**: `exp(-2.0 * elbow_pos^2)` (最大 +1.0)

3.  **速度惩罚 (Velocity Penalty)**:
    - **公式**: `-0.1 * |velocity|`
    - **说明**: 惩罚关节角速度，鼓励系统保持静止，防止高频振荡。

### 3.2 惩罚与重置 (Reset)

- **重置条件**: 任意关节角度绝对值超过 `reset_dist` (默认 3.0 弧度，约 172度)。
- **死亡惩罚**: `-2.0`
  - 如果触发重置，当前步的奖励直接被覆盖为 -2.0。

### 3.3 理论最高分

- **单步最高分**: 
  - 存活(1.0) + Shoulder完全直立(1.0) + Elbow完全直立(1.0) - 静止(0.0) = **3.0 分**
- **单回合最高分 (Max Episode Reward)**:
  - 3.0 分/步 * 500 步 = **1500 分**

## 4. 训练观察指南

- **初期 (Reward < 0)**: Agent 还在探索，经常触发死亡惩罚(-2.0)。
- **中期 (Reward 100 ~ 500)**: Agent 学会了甩起来，但还不够稳，或者保持时间不够长。
- **后期 (Reward > 1000)**: Agent 能够快速甩起并稳定悬停在直立位置。

## 5. 如何调整

如果发现训练效果不佳，可以按以下方向调整：
- **摆动无力**: 稍微增加 `maxEffort` (如 5.0 -> 8.0)。
- **疯狂抖动**: 增加速度惩罚权重 (如 0.1 -> 0.2) 或减小 `maxEffort`。
- **收敛太慢**: 检查 PPO 的 `learning_rate`。
